Квадратная матрица.
Пользователь вводит строки содержащие цифры разделенные запятыми. Например "1,2,3,4"
Пустая строка означает окончание ввода.
После окончания ввода одной строки, запрашивается следующая и т.д.
Две пустых строки подряд означает, что пользователь закончил ввод данных.
Необходимо проверить что введенная матрица является квадратной (количество строк равно количеству столбцов) 
Если матрица является квадратной, то посчитать ее определитель.
